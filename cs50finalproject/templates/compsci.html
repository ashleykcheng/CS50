Harvard CS50x

CS50 is an entry-level course taught by David J. Malan at Harvard University. CS50x is a self-paced class that includes recorded lectures from Harvard and the same problem sets Harvard students are assigned. Languages taught include C, PHP, and JavaScript plus SQL, CSS, and HTML. Problem sets are inspired by real-world domains of biology, cryptography, finance, forensics, and gaming. Students learn how to think algorithmically and solve programming problems efficiently while exploring concepts like abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development. This website was created as the final project of this course.
